# 장중 실시간 거래대금 순위 (k-rank)

KIS 거래량순위 API(거래금액순)로 **지금 이 순간**의 거래대금 상위 종목을 조회한다.
KRX 일별매매정보(EOD)와 달리 장중 실시간 값이며, 응답이 빠르다(토큰 캐시 시 ~0.1초).

```bash
cd "${CLAUDE_SKILL_DIR}/../../.." && export PATH="$HOME/.local/bin:$PATH" && uv run python "${CLAUDE_SKILL_DIR}/scripts/run.py"
```

## 사용 구분
- **장중(09:00~15:30)에 "지금 실시간"** 질문 → 이 스킬(k-rank).
- **전일 확정 종가·거래량 기준** → k-movers.
- 장 운영시간이 아니면 직전 값이 나오거나 빈 결과일 수 있다(그 경우 k-movers 권장).

## 출력 활용
- 표를 그대로 보여준다(순위·종목·현재가·등락률·거래대금·거래량·시장).
- 상위에 특정 테마(반도체·방산 등)가 몰리면 한 줄로 짚어준다.
- 특정 종목에 관심을 보이면 **k-analyze**(종목 분석)로 이어가도록 제안한다.
