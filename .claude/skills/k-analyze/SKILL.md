# 종목 풀 분석 (k-analyze)

사용자가 지정한 종목을 분석한다. 아래 명령을 실행하면 마크다운 분석 결과가 출력된다.

```bash
cd "${CLAUDE_SKILL_DIR}/../../.." && export PATH="$HOME/.local/bin:$PATH" && uv run python "${CLAUDE_SKILL_DIR}/scripts/run.py" $ARGUMENTS
```

## 출력 활용
- 출력은 이미 사람이 읽을 마크다운이다. **그대로 사용자에게 보여준다.**
- 마지막에 **한 줄 매매 코멘트**를 덧붙인다 (예: "중·장기 매수 신호이나 초단기는 관망. ATR 손절은 표준 2.0x 기준 설정 권장").
- 점수가 모두 관망(Hold)이면 "뚜렷한 진입 신호 없음"이라고 솔직히 말한다.
- 데이터 기준일이 오늘이 아닐 수 있다(직전 거래일). 기준일을 함께 안내한다.

## 주의
- 투자 판단의 근거 자료일 뿐, 매매 권유가 아니다. 손절가는 반드시 함께 제시한다.
