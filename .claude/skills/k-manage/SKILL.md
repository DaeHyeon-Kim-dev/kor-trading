# 보유 포지션 관리 (k-manage)

평단가를 받아 현재 보유 종목의 액션(보유/추가/익절/손절)을 조언한다.

```bash
cd "${CLAUDE_SKILL_DIR}/../../.." && export PATH="$HOME/.local/bin:$PATH" && uv run python "${CLAUDE_SKILL_DIR}/scripts/run.py" $ARGUMENTS
```

## 출력 활용
- 평가손익 + 판단(보유/추가/익절/손절) + 근거 + 권장 손절선을 그대로 보여준다.
- 판단이 "전량 익절"/"손절"이면 **추세 이탈** 신호이므로 우선순위 높게 전달한다.
- "일부 익절"은 과열 신호 — 전량이 아니라 분할 대응임을 명확히 한다.
- 권장 손절선은 추세 유지의 마지노선(20일선/ATR 기준)임을 안내한다.
- 평가 기준일이 직전 거래일일 수 있으니 날짜를 함께 안내한다.
- 최종 판단·책임은 사용자에게 있음을 짧게 명시한다.
