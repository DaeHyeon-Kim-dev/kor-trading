# 지금 매수할 만한 종목 (k-screen)

후보 종목을 스캔해 매수 셋업이 잡힌 종목만 랭킹한다. 수십 종목 분석이라 1~2분 걸릴 수 있으니 오래 걸리면 백그라운드로 실행한다.

```bash
cd "${CLAUDE_SKILL_DIR}/../../.." && export PATH="$HOME/.local/bin:$PATH" && uv run python "${CLAUDE_SKILL_DIR}/scripts/run.py"
```

## 출력 활용
- 시장 레짐 + 셋업 매칭 종목 표(셋업·강도·과거기대값·진입/손절/목표)를 그대로 보여준다.
- **시장 레짐이 약세면** 롱 신호를 신중히(비중 축소·엄격 손절) 다루라고 강조한다.
- "과거 기대값"은 백테스트(상승장·거래비용 미반영) 참고치임을 명시한다. 미검증 셋업은 할인.
- 사용자가 특정 종목을 고르면 `/k-analyze <종목>`로 수급·공시까지 깊이 본 뒤 진입을 판단하도록 이끈다.
- 셋업 매칭 종목이 없으면 "지금은 살 만한 자리 아님(관망)"이라고 솔직히 전한다.
- 매수 권유가 아니라 후보 제시임을 명시한다.
